A Quick Manual Check (If the script doesn't instantly solve it)If you run the script, reboot, and the Wi-Fi icon still shows an error, AVG's virtual network adapter driver might still be clinging to your physical hardware profile. You can drop it manually in seconds:Press Win + R, type ncpa.cpl and hit Enter (this opens Network Connections).  Right-click your Wi-Fi adapter and select Properties.  Look at the itemized list under "This connection uses the following items:"If you see anything with "AVG" or "CCleaner" in the name (like AVG Network Filter Driver), select it and click Uninstall.While you are in there, double-click Internet Protocol Version 4 (TCP/IPv4) and ensure both options are set to "Obtain an IP address automatically".  Once the uninstallation script cleans out the software and this script flushes the plumbing, your computer should instantly grab an IP address from your router again.
